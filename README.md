# Numbers to 200 — Inellia Educational Module

## Quick Start

```bash
npm install
npm run dev
```

---

## 🔊 Audio Setup — ElevenLabs

### Step 1 — Create your .env.local file
```bash
cp .env.local.example .env.local
```

### Step 2 — Open `.env.local` and paste your API key
```
VITE_ELEVENLABS_API_KEY=sk_your_actual_key_here
```
Get your API key: https://elevenlabs.io → Log in → Profile icon → API Keys

### Step 3 — (Optional) Change the voice
```
VITE_ELEVENLABS_VOICE_ID=Xb7hH8MSUJpSbSDYk0k2
```
The default is **Alice** (Xb7hH8MSUJpSbSDYk0k2) — a clear, engaging educator voice.
Browse all voices: https://elevenlabs.io/voice-library

### Step 4 — Restart the dev server
```bash
npm run dev
```

> ⚠️ Without a key the app still works — audio is silently skipped.

## Pre-generate audio assets (recommended for production)
```bash
npm run generate-audio   # hits ElevenLabs, saves .mp3s, updates audioMap.js
npm run clean-audio      # removes orphaned .mp3 files
```

---

## Where to put things

| What | Where |
|------|-------|
| ElevenLabs API Key | `.env.local` → `VITE_ELEVENLABS_API_KEY` |
| Voice ID | `.env.local` → `VITE_ELEVENLABS_VOICE_ID` |
| Story images | `public/assets/images/` |
| Pre-generated audio | `public/assets/audio/` |

## Project Structure

```
src/
  components/
    phases/        WonderPhase, StoryPhase, SimulatePhase, PlayPhase, ReflectPhase
    game/          QuestionCard, AnswerOption, ScoreBar

    shared/        Mascot, SpeechBubble, PhaseNav, AudioToggle, Confetti
  utils/

    questionEngine.js
    placeValue.js
  data/
    questions.js   100-question bank
public/
  assets/
    images/        Story scene images
    audio/         Pre-generated .mp3 files
```
